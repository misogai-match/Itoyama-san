概要

MetaFusion株式会社（代表：原口信克）が手掛けるサフラン由来の医薬用クロシン抽出・精製事業の初回説明会。千葉さんが事業の経緯・技術・収益構造を説明し、糸山側は事業への関心を示しつつ、バリュエーション等の確認事項を宿題として持ち帰った。

事業の概要

経緯（千葉さん説明）

  • 6年前に農業を志し、水耕栽培の技術者・中島氏と出会う（10年以上の付き合い）
  • 中島氏がサフランの室内栽培に成功 → 通常年1回の収穫を年6回に
  • メシベからクロシン（医薬品成分）を抽出し、さらにクロシン5（高純度精製品）まで一貫製造
  • 世界で農家がクロシン5まで製造している例はおそらくない
  • 昨年、薬の製造まで完結。5年かけてようやくゴールが見えた段階

技術的優位性

  • 中島氏の抽出技術で過去にない高純度を実現 → 逆に製薬会社側の水溶化に時間がかかるほど
  • 水耕栽培技術はアメリカ特許取得済み
  • ピクロクロシン（アルツハイマー薬の候補成分）も分離済み（微量）
  • カロリンスカ研究所（ノーベル賞選考機関）にて、メルク社・中島氏と共同法人を5〜6月に設立予定 → アルツハイマー薬の認可を目指す

取引先

  • ドイツ・メルク社が購入先。契約書の雛形は受領済み
  • メルク社は世界中からクロシンを年間8,000億〜1兆円購入している
  • メルク社の要望：年間6kgの安定供給
  • 現在、水溶化テスト中 → あと約1週間で完了 → 正式発注書が出る
収益構造

単価

仕入→販売の利益計算（千葉さん説明）

例：メシベ1kgを仕入れた場合
仕入：1kg × 4,000円/g = 400万円
1kg = 100万mg → クロシン含有率1%で = 1万mg
売上：1万mg × 2万円/mg = 2億円
抽出工場の経費を入れても1,000万円以下
→ 粗利率 95%以上

会議での質疑・やり取り

糸山さんの反応

  • 「投資家ではなく事業家」— リターンよりビジネスとしての面白さに関心
  • 「千葉さんの今やってるビジネスが楽しそうだなと思って」
  • IPOの可能性を質問 → 千葉：「これから考えている。サウジアラビアなど複数パターン検討中」
  • ファンドの話は？ → 千葉：「まだしていない。まず個人投資家から。原口さんにファンド会社の付き合いはある」

磯貝の質問

  • バリュエーションの算定方法を質問 → 千葉：「原口社長の判断で決めている。確認して回答する」
  • 磯貝の見解：競合不在でマルチプル法は使えない → DCF法で将来キャッシュフローベースの算定が必要
  • 蓋然性の整理を宿題として持ち帰り

千葉さんの将来構想

  • サフランの利益で廃校跡地を購入 → 農業 + 陸上養殖（ウニ、6月完成予定） + 「逃れの街」構想
  • 「サフランは手段」— 高収益を活かした地域再生が最終目標
リスク・要確認事項
  • 売上実績ゼロ — 設立3年、資本金8,000万円、月600万円のランニングで赤字継続中
  • バリュエーション未算定 — 企業価値の根拠なく第三者割当を実施中
  • 水溶化テスト未完了 — 「あと1週間」と言われているが、薬事プロセスは遅延リスクあり
  • ピッチ資料と口頭説明の乖離 — 東大連携・中外製薬・調達量等の裏付け確認が必要
  • 収益計画の飛躍 — FY2027 5億→FY2030 825億は極めて楽観的。前提条件の精査必要
  • メルク社との契約 — 「契約書雛形は来ている」が正式発注書はまだ
  • 原料調達リスク — 中国・インドのルートはサンプル段階。現地未訪問（今月渡航予定）
  • 会社構造 — MetaFusion（原口代表、メシベ仕入会社）と千葉さんの工場は別法人。関係性の整理必要
